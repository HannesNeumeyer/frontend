SEO

SEO steht für Search Engine Optimization (Suchmaschinenoptimierung) und bezeichnet Maßnahmen, mit welchen Websites im Ranking von Suchmaschinen weiter vorne erscheinen. Sucht ein Nutzer bei Google beispielsweise nach bestimmten Schlagwörtern, dann wird er mit einer hohen Wahrscheinlichkeit auf eines der Suchergebnisse auf der ersten Seite klicken. Nur die wenigsten schauen sich auch Suchergebnisse auf der zweiten oder dritten Seite an und noch weniger die der folgenden Seiten. SEO ist also für das gefunden werden einer Website von entscheidender Bedeutung und ist somit ein essentieller Bestandteil des Web‑Marketing. Für eine SEO optimierte Website sollte man folgende Dinge beachten:

Beschaffenheit einer Website

  • sicher (https)
  • schnelle Ladezeit
  • mobil optimiert
  • klare URLs

Inhalt einer Website

  • guter Titel im title‑tag
  • Beschreibung im meta‑tag
  • alt‑attribut bei img verwenden
  • h‑tags für die Gliederung

Kontakte einer Website

  • andere Websites verlinken auf die eigene
  • link zur Website auf Social‑Media Seiten

Eine ausführliche Beschreibung findet man im SEO Style‑Guide von Google.

Zu guter letzt sei noch gesagt, dass es einfach eine Weile dauert, bis eine Website im Ranking steigt. Die meisten Websites brauchen ca. 2-3 Jahre bis sie auf den vorderen Plätzen einer Suchmaschine landen.
