What The Face

Beim Entwickeln von Websites gibt es neben dem eigentichen Schreiben von Code noch einige andere Bereiche, von denen man zumindest einmal gehört haben sollte. Vor allem für Entwickler im Team führt kein Weg an Git vorbei. Git ist eine Versionsverwaltung, die das gemeinsame Arbeiten an einer Website erleichtert und mit der man jeden Zeitpunkt der Entwicklung nachvollziehen kann. Auch SEO sollte jedem Entwickler ein Begriff sein. Was bringt einem die schönste und beste Website der Welt, wenn keiner sie findet. SEO ( Search Enginge Optimization ) sorgt dafür, dass eine Website im Ranking der Suchmaschinen weiter oben landet und somit besser gefunden wird. Zu guter letzt steht das Thema Datenschutz. Auch wenn das Thema an sich keinem Entwickler wirklich Spaß macht, sollte man über rechtliche Dinge im Web Bescheid wissen.
