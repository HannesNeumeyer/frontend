- Es geht um mehr als "nur" den Inhalt. -

Gibt es im Web wirklich so viele Online-Sehtests oder ist man gerade einfach mal wieder auf einer Website mit winziger Schrift gelandet? Ok, vielleicht ist das tatsächlich ein Sehtest. Dann machen die kleinen Buchstabene durchaus Sinn, bei denen man sich so anstrengen muss um sie zu entziffern. Andernfalls hat sich der Entwickler der Website über das Text-Layout wohl einfach keine Gedanken gemacht und ihm war nicht bewusst, dass Nutzer mit hoher Wahrscheinlichkeit die Website gleich wieder verlassen werden, da sie einfach keine Lust auf einen weiteren Sehtest haben.
