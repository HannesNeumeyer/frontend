• Schriftgröße

    Wie einleitend schon erwähnt ist die Größe der Schrift entscheidend für eine positive Erfahrung des Nutzers mit der Website. Sie sollte dabei nie zu klein sein und auch entsprechend des Endgeräts (Smartphone, Tablet, PC) angepasst werden.

    CSS Code: font-size: 20px;

    Testwort

    font-size: 20px

    Testwort

    font-size: 40px

  • Schriftstärke

    Die Stärke der Schrift ist vor allem dazu da, um bestimmte Teile im Text hervorzuheben. Der Nutzer soll so schnell erkennen können, was besonders wichtig ist. Außerdem dient sie zur Untergliederung von inhaltlichen Abschnitten z.B. für Überschriften.

    CSS Code: font-weight: 700;

    Testwort

    font-weight: 400

    Testwort

    font-weight: 700

  • Schriftart

    Die Schriftart bietet viel Gestaltungsraum, sollte aber gut gewählt sein. Eine Schriftart die besonders geschmückt ist und schön aussieht, ist für einen längeren Fließtext vielleicht nicht so gut geeignet wie eine einfache Schrift.

    CSS Code: font-family: Lato;

    Testwort

    font-family: Lato

    Testwort

    font-family: Lobster

  • Schriftfarbe

    Die Schriftfarbe dient ebenso wie die Schriftstärke zum hervorheben von Text. Aber auch hier sollte man eher genau darauf achten, was das Ziel des Textes ist. Denn zu viel oder grelle Farbe kann vom eigentlichen Inhalt des Text ablenken.

    CSS Code: color: #333;

    Testwort

    color: royalblue

    Testwort

    color: crimson

  • Ausrichtung des Text

    Mit der Ausrichtung des Text entscheidet man über die Position des Textes. Überschriften stehen meistens mittig, während Fließtext oft von links ausgerichtet wird. Aber auch ob der Text im Blocksatz dargestellt werden soll, kann mit der Ausrichtung bestimmt werden.

    CSS Code: text-align: center;

    Testwort

    text-align: left

    Testwort

    text-align: center

  • Zeichenabstand

    Der Zeichenabstand beschreibt den Abstand zwischen den einzelnen Buchstaben. Ein bisschen Leerraum zwischen den Buchstaben macht das Lesen oft angenehmer. Auch bei Überschriften macht ein Zeichenabstand oft Sinn.

    CSS Code: letter-spacing: 2px;

    Testwort

    letter-spacing: 2px

    Testwort

    letter-spacing: 6px

  • Zeilenabstand

    Ebenso wie der Zeichenabstand dient der Zeilenabstand um Leerraum zu schaffen, allerdings zwischen den einzelnen Zeilen. Auch hier gilt, dass Leerraum angenehmer wirkt.

    CSS Code: line-height: 1.9;

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Sint soluta odio consectetur ipsum consequatur eos obcaecati ullam.

    line-height: 1.5

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Sint soluta odio consectetur ipsum consequatur eos obcaecati ullam.

    line-height: 1.9
