HTML img vs CSS background‑image

Man kann sowohl mit dem HTML‑tag img als auch mit der CSS Eigenschaft background-image Bilder im Web darstellen. Nur wann sollte man welche Variante wählen?
 Grundsätzlich gilt die Faustregel: Ist das Bild ein Teil des Inhalts, dann sollte man das Bild über den HTML‑tag img einbauen und zusätzlich das Bild mit dem alt Attribut beschreiben. Dies begünstigt die Erreichbarkeit bei Suchmaschienen wie Google.
Bilder die ausschließlich zu Design Zwecken verwendet werden können auch über die CSS Eigenschaft background‑img in eine Website integriert werden.
