Aplication Programming Interface (API)

Eine API ist eine Programmierschnittstelle, die einer Website eine Anbindung an andere Systeme und so den Austausch von Daten ermöglicht. Google‑Maps Karten können beispielsweise über die Google API in die eigene Website eingebaut werden. Mit den Wetterdaten von wunderground.com könnte man zum Beispiel seine eigene Wetter‑App entwickeln und bräuchte dazu nicht einmal ein Backend oder eine Datenbank.
